# 腾讯云通用组件

## **简介**（Introduction）

`文本内容安全`

### **安装**

在 Maven 工程中使用

```
    <dependency>
        <groupId>com.zhengcheng</groupId>
        <artifactId>zc-tencentcloud-spring-boot-starter</artifactId>
    </dependency>
```

## 初始化

在配置文件中增加：

```
tencentcloud.secret-id=您的secretId
tencentcloud.secret-key=您的secretSecret
tencentcloud.region=ap-guangzhou
```
